# Jorant Design System

Design system build for publications.vd.ch

## Run locally

```
$ npm install
$ npm start
```

## Deploy

To deploy the design system on https://dsi-vd.github.io/joran-design-system/, use the following commands:

```
$ npm install
$ npm run deploy
```
